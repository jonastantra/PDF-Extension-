Privacy Policy 100% Local

Ultimate PDF: All-in-One Editor - Chrome Extension

Last updated: December 2024

Summary: Ultimate PDF: All-in-One Editor does NOT collect, store, or transmit any personal data or file contents. All processing happens 100% locally on your device.

1. Introduction

This Privacy Policy explains how Ultimate PDF: All-in-One Editor ("the Extension", "we", "our") handles user data. We are committed to protecting your privacy and being transparent about our practices.

2. Data Collection

We do NOT collect any data. Specifically:

  • No personal information - We don't collect names, emails, or any identifying information
  • No file contents - Your PDFs, images, and documents are never uploaded or transmitted
  • No browsing history - We don't track which websites you visit
  • No analytics - We don't use Google Analytics or any tracking services
  • No cookies - We don't use tracking cookies

3. Data Processing

All file processing occurs entirely on your device:

  • PDF conversion, merging, splitting, rotating, and compression happen in your browser
  • No files are ever sent to external servers
  • No internet connection is required for file processing (except for the "Webpage to PDF" feature which requires loading the webpage)
  • Once you close the extension, no file data is retained

4. Local Storage

The extension stores minimal data locally on your device:

  • Language preference - Your selected interface language
  • Tool settings - Your preferred options (quality, format, etc.)

This data:

  • Is stored only on your device using Chrome's local storage API
  • Is never transmitted to any server
  • Can be deleted by clearing the extension's data or uninstalling it

5. Permissions Explained

The extension requests the following permissions:

  • sidePanel - To display the extension interface in Chrome's side panel
  • storage - To save your preferences locally on your device
  • activeTab - To capture the current webpage for the "Webpage to PDF" feature
  • tabs - To get the title and URL of the current page for naming saved PDFs
  • scripting - To execute the webpage capture functionality

These permissions are used only for the stated purposes and do not enable any data collection.

6. Third-Party Libraries

The extension includes the following open-source libraries, which are bundled locally:

  • PDF-lib - PDF manipulation (MIT License)
  • PDF.js - PDF rendering by Mozilla (Apache 2.0 License)
  • jsPDF - PDF creation (MIT License)
  • JSZip - ZIP file handling (MIT License)

These libraries run entirely locally and do not make any network requests.

7. Data Security

Your data security is ensured by:

  • 100% local processing - files never leave your device
  • No external server connections for file operations
  • Manifest V3 compliance - Chrome's most secure extension format
  • Strict Content Security Policy implementation

8. Children's Privacy

The extension does not collect any data from anyone, including children under 13. Since no data is collected, there are no special provisions needed for children's privacy.

9. Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date at the top of this document. Significant changes will be communicated through extension updates.

10. Your Rights

Since we don't collect any personal data, there is no personal data to access, modify, or delete. You maintain full control over your files at all times.
